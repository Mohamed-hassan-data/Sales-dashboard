# Sales Analysis Dashboard - Excel Project

## 📌 Project Overview
This project presents an interactive Sales Analysis Dashboard created using Microsoft Excel.  
The dashboard helps analyze sales performance, profit, quantity sold, discount impact, and customer distribution across states.

The purpose of this dashboard is to transform raw sales data into meaningful business insights through data visualization and KPI tracking.

---

## 🎯 Objectives
- Monitor overall sales performance
- Analyze profit by category and subcategory
- Track sales trends and quantity sold
- Identify top and bottom performing products
- Analyze customer distribution by state
- Provide interactive filtering for better decision-making

---

## 📊 Dashboard Features

### KPI Cards
The dashboard includes the following Key Performance Indicators (KPIs):

- Total Sales
- Total Profit
- Total Quantity
- Average Discount
- Profit Margin

---

### Visualizations
The dashboard contains:

- Donut charts for:
  - Profit by Category
  - Sales by Category
  - Quantity by Category

- Bar charts for:
  - Profit by Subcategory
  - Top 10 Products by Sales
  - Bottom 10 Products by Sales

- Combined chart for:
  - Sales & Profit by Subcategory

- Geographic Map:
  - Total Customers by State

## 🔍 Key Insights
- Technology category generated the highest sales.
- Office Supplies contributed significantly to total quantity sold.
- Some products generated high sales but low profit margins.
- Customer distribution varies significantly across states.

## 🚀 How to Use
1. Open the Excel dashboard file.
2. Use slicers to filter data by:
   - Year
   - Quarter
   - Category
3. Interact with charts for dynamic analysis.
